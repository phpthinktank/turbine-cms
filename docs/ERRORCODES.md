# Turbine Error codes

## Runtime 5xx

| code | message                                |
| ---- | -------------------------------------- |
| 0    | Any Error                              |
| 501  | Class not found                        |
| 502  | Invalid class                          |

## Config 10xx

| code | message                                |
| ---- | -------------------------------------- |
| 1001 | Environment not found                  |
| 1002 | Unknown node type                      |
| 1003 | Unable to determine node               |
